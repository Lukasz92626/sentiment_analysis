# IMDb Sentiment Analysis for Movie Reviews

Sentiment classification of IMDb movie reviews using Machine Learning.

## Features

- 3 vectorization methods: TF-IDF, Count Vectorizer, Hashing Vectorizer
- 2 classifiers: Logistic Regression & Naive Bayes
- Model comparison & persistence (pickle)

## Dataset

[IMDb movie reviews dataset](https://ai.stanford.edu/~amaas/data/sentiment/) (50,000 labeled samples: 25k train / 25k test)
